### ♈ AstroDate — Deep Dive

> A next-generation **astrology-powered dating platform** that goes beyond superficial matching. AstroDate uses real-time planetary position data, birth chart generation, and a custom **ML compatibility engine** to pair users based on synastry — the study of how two birth charts interact.

**Core Features:**
- 🌌 **Birth Chart Engine** — Generates full natal charts using birth date, time & location via astronomy APIs
- 🔮 **Synastry Scoring** — Proprietary ML model scores compatibility across Sun, Moon, Venus, Mars & Rising signs
- 💬 **Astro-Chat** — Personalized conversation starters generated from shared planetary aspects
- 🗺️ **Compatibility Map** — Visual radar chart of elemental & modal compatibility between matches
- 🔔 **Transit Alerts** — Notifies users when planetary transits create optimal connection windows
